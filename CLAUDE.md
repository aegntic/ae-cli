# aedex (aegntic) — project notes for Claude Code

## Design System
Always read **DESIGN.md** before any visual or UI decision.
All fonts, colors, spacing, and aesthetic direction are defined there (dark
editorial + electric azure `#2E9BFF` + Space Grotesk/Inter + JetBrains Mono).
Do not deviate without explicit user approval. In QA, flag any code that does
not match DESIGN.md (e.g. hardcoded `bg-white`/`border-black` from the
deprecated light "Aegntic Toy" system — those must use dark tokens).

See also: `AGENTS.md`, `CONSTITUTION.md`, `docs/SECURITY.md`.
