# YouTube — checkpoint scripts

One script per checkpoint beat in `docs/roadmap.md`. Naming: `NN-checkpoint-name.md` (NN = phase number).

Each script:
- **Title** (≤60 chars, hook + payoff)
- **Cold open** (first 8s — the moment of proof)
- **Setup** (what we're about to see, 1 sentence)
- **Demo / build** (screen capture + voiceover beats)
- **Fork-in-road** (the decision this checkpoint turned on — link its ADR)
- **CTA** (next checkpoint)

Capture motion video + voiceover at the checkpoint. Vertical clips for shorts cut from the cold open.
